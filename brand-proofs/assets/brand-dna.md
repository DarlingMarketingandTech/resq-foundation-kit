# ResQ Brand DNA Audit (Proof Baseline)

> Captured for `brand-proofs/` mockups only. Sources: [resqorganics.com](https://resqorganics.com/), [resqorganics.com/shop](https://resqorganics.com/shop/), Amazon brand store (ResQ Organics for Pets & People).

## Human / People line (current site)

| Attribute | Observation |
|-----------|-------------|
| Voice | Warm, family-owned ("From our family to yours"), ingredient-forward |
| Layout | Text-heavy homepage, category grid on shop, older WooCommerce theme |
| Categories | Skin Care Men/Women/Babies, CBD Products — maps to People + CBD lanes |
| Imagery | Mixed quality; product photos vary; lifestyle minimal |
| Risk | Legacy copy uses disease/treatment claims — **not carried into proofs** |

## Pets + Amazon presence

| Attribute | Observation |
|-----------|-------------|
| Imagery | White-background packshots, infographic secondary images, consistent listing layout |
| Feel | More polished DTC / marketplace-ready than current website |
| Categories | CBD for Pets, Pet Treats, Skin Care for Pets — maps to Pets + CBD lanes |
| Gap | Amazon look ≠ website look → harmonization problem the proofs solve |

## Shared equity (keep in all directions)

- **Name:** ResQ Organics
- **Ingredients story:** Manuka honey, aloe, organic/natural positioning
- **Trust:** Non-GMO, paraben-free, family-safe framing (claim-safe wording in proofs)
- **Audiences:** People (human skincare), Pets, CBD (isolated lane)
- **Giving back:** Optional mention in footer — not hero focus for proofs

## Proof copy rules (compliance-safe)

- No disease names, cure language, or FDA-style drug claims
- Use: "nourish," "support," "daily routine," "organic ingredients"
- CBD: age/state disclaimers as placeholder slots only

## Image strategy in proofs

- CSS/SVG product placeholders (no external image dependency for offline viewing)
- Placeholder labels reference real SKU names from catalog import
- Direction C uses packshot-style frames; Direction B uses distinct photo treatments per sub-brand
