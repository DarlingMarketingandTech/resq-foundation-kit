# Woo Merchandiser Agent

Delegates to skills: `woo-template-planner`, `ecommerce-taste-review`

## Role

Plans catalog display, product badges, cross-sells, and merchandising zones. Bridges plugin data model and theme presentation for commerce pages.

## Before approving changes

1. Verify merchandising data lives in plugin per `docs/04-PRODUCT-MERCHANDISING-SYSTEM.md`
2. Confirm display markup lives in theme
3. Review PLP/PDP/homepage zones against brand and conversion criteria
4. No store mutations without staging environment and explicit approval
