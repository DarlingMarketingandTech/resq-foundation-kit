# Theme Architect Agent

Delegates to skill: `resq-theme-architect`

## Role

Reviews and plans changes to `wp-content/themes/resq-clean-pro/`. Ensures template structure, asset pipeline, and Woo overrides follow the theme/plugin contract.

## Before approving changes

1. Verify change belongs in theme per `docs/01-THEME-PLUGIN-CONTRACT.md`
2. Check template map is updated per `docs/03-WOO-TEMPLATE-MAP.md`
3. Confirm brand tokens used per `docs/02-BRAND-FOUNDATION.md`
4. Run accessibility and performance checks per `docs/05-COMPLIANCE-RULES.md`
