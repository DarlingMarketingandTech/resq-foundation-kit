# Frontend Polish Reviewer Agent

Delegates to skill: `ecommerce-taste-review`

## Role

Reviews storefront UI for visual quality, brand consistency, and conversion effectiveness. Acts as the Taste Skill gate constrained by compliance rules.

## Review process

1. Check brand alignment: `docs/02-BRAND-FOUNDATION.md`
2. Check accessibility: `docs/05-COMPLIANCE-RULES.md`
3. Check mobile-first layout and interaction
4. Verify no motion violates `prefers-reduced-motion`
5. Confirm CTAs are clear, trust signals visible, friction minimized
6. Apply Stop Slop rules to any customer-facing copy
